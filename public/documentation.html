myflix API

Objective

The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Objective

  • Return a list of ALL movies to the user
  • Return data (description, genre, director, image URL, whether it's featured or not) about a single movie by title to the user
  • Return data about a genre (description) by name/title
  • Return data about a director (bio, birth year, death year) by name
  • Allow new users to register
  • Allow users to update their user info
  • Allow users to add a movie to their list of favorites
  • Allow users to remove a movie from their list of favorites
  • Allow existing users to deregister.

ENDPOINT DOCUMENTATION:

Request URL HTTP Method Response
Return a list of ALL movies to the user /movies GET a JSON object holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/Title GET a JSON object holding data about a single movie containing the data above
Return data about a Genre (Description) by Name (e.g., “Thriller”) /movies/Genre/Name GET a JSON object holding data about genre name and its description
Return data about a Director (Bio, Birth year, Death year) by Name /movies/Director/Name GET a JSON object holding data about director such as name, bio, date of birth
Allow new users to register /users POST a JSON object holding information about a newly created user account
Allow users to update their user info (Username) /users/Username PUT a JSON object holding updated information of a user account
Allow users to add a movie to their list of favorites (showing only a text that a movie has been added) /users/Username/FavoriteMovies/[ObjectId] POST a JSON object containing information of a user account with movie title added
Allow users to remove a movie from their list of favorites (showing only a text that a movie has been removed) /users/Username/FavoriteMovies/[ObjectId] DELETE a JSON object holding updated data with a requested movie deleted from the favorites list
Allow existing users to deregister (showing only a text that a user email has been removed) /users/Username DELETE a text message indicating a user account was deleted
